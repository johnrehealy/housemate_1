# Housemate

This repository is the canonical source of truth for the Housemate product, brand, architecture, and roadmap.

## What Lives Here

- `CLAUDE.md`: operating instructions and top-level project context
- `docs/`: canonical living product docs
- `tasks/`: working task log and lessons learned
- `references/palette/`: active visual reference inputs
- `references/legacy-docx/`: older `.docx` source artifacts kept for reference only
- `scripts/`: utility scripts used to maintain or migrate source artifacts

## Source Of Truth Rule

The living truth is Markdown in this repo, not the legacy `.docx` files.

Read first:

1. `CLAUDE.md`
2. `tasks/todo.md`
3. `docs/00-source-of-truth.md`
4. the rest of `docs/`

## Current Goal

Move from a broad strategy corpus to a clear minimum lovable product, then convert that scope into executable work tracked in Linear.

## Important Constraint

Linear tracks execution. It does not replace the product brain in this repo.
