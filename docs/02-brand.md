# Housemate Brand

Last updated: 2026-04-22

## Brand Character

Housemate should feel like the brilliant friend who happens to know everything about your home: warm, assured, quietly excellent, and never performative.

## Voice

The voice is:

- warm without being familiar
- professional without being cold
- calm under pressure
- direct and orienting when things go wrong

The voice is not:

- robotic
- alarmist
- jargon-heavy
- performatively cozy
- explicit about AI unless the context requires it

## Core Messaging

- Tagline: `Because homes don't manage themselves.`
- Primary CTA: `Consider it done.`
- One-line promise: `The level of home care that used to require a staff.`

## Communication Norms

- Emotion before action.
- Earn the right to the next step.
- Hold the thread and close loops.
- Ask for one thing at a time when clarification is needed.
- Calm is the default; direct urgency is reserved for true safety cases.
- Acknowledge, do not over-explain, and always provide a path forward.

## Design North Stars

- Superhuman
- Linear

Interpretation:

- minimalist, information-dense, and deliberate
- premium without feeling ornate
- sharp but not cold
- generous whitespace and disciplined hierarchy

## Typography

- Primary typeface: Geist Variable
- Use one type family across the product and create hierarchy through weight, scale, and color

## Current Palette Truth

Canonical brand version: `Ember Signal · Dusk` v1.5

### Sunset Scale

- `#FF4E7D` Sunset Vivid
- `#F03B65` Sunset Subtle
- `#E8305A` True Primary
- `#CC2848` Primary Hover
- `#FF7040` Ember Warm

### Neutrals

- `#FAF8F7` page background
- `#FFFDFC` card
- `#ECE8E6` border
- `#B8B0AD` tertiary text
- `#8A8380` muted text
- `#1A1614` foreground

### Dusk Surfaces

- `#161214` page
- `#1F1B1D` card
- `#262123` elevated
- `#2A2528` border

### Agent Gradient

- `#FF4E7D -> #FF7040`
- Reserved for agent-present contexts only
- Use once per screen, not as decorative wallpaper

## Palette Rules

- Warm-tinted neutrals are non-negotiable.
- Do not introduce cool-gray neutrals.
- Cards should use `#FFFDFC`, not pure white.
- `#FFFFFF` is valid for on-sunset text.
- The undertone rule matters: every neutral should feel native to the sunset family.
